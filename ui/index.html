Configurações

Automático escolhe a melhor opção disponível na ordem AV1 hardware → HEVC hardware → H.264 hardware → H.264 software. Use uma escolha manual só se souber o que está fazendo.
+
Marque os modificadores desejados (Ctrl, Shift, Alt, Win) e escolha uma tecla principal. Exemplos: Pause ou Ctrl+Shift+G.
O NoOBS sobe automaticamente quando você loga no Windows.
Permite ao NoOBS usar o ícone na bandeja do sistema. Com esta opção ativa, o botão X da janela manda pra bandeja em vez de encerrar — clique no ícone pra reabrir, ou use o menu do ícone pra fechar de verdade.
Ao começar uma gravação, a janela do NoOBS é minimizada — útil para gravar a tela sem a própria janela aparecer no vídeo. O atalho global continua funcionando pra parar.
Mostra uma notificação do Windows quando começa ou termina uma gravação. Só faz sentido quando a janela é minimizada automaticamente — caso contrário o estado da gravação já é visível na própria janela.
